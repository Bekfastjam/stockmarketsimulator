# Stock Market Simulator

A simple command-line stock market simulator written in Python.

## Features

- Simulates basic buying and selling of stocks
- Tracks your portfolio and cash balance
- Randomly changes stock prices each round
- Easy to run and modify

## Getting Started

### Prerequisites

- Python 3.x

### How to Run

1. Clone this repository or download the script.
2. Open a terminal and navigate to the project folder.
3. Run:

   ```
   python stock_market_simulator.py
   ```

### How to Play

- Choose to buy or sell stocks each round.
- Enter the stock symbol and quantity.
- Watch your balance and stock prices change after every action.
- Type `quit` to exit the simulator.

## Example

```
Your portfolio:
AAPL: 0 shares (Price: $150)
GOOG: 0 shares (Price: $2800)
TSLA: 0 shares (Price: $700)
Cash balance: $10000

Available actions: buy, sell, quit
What would you like to do? buy
Enter stock symbol: AAPL

```
